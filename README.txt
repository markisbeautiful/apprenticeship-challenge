Installation (necessary once before the first execution)


1. Install Python 3.X (for example: https://realpython.com/installing-python/)
	
	Windows: Install requests:

		1.1. Run command prompt (Press Win + R keys, type 'cmd' and press Enter)

		1.2. Type 'python -m pip install requests' and press enter



Running the script (internet connection required!)

1. Run command prompt(Windows) / terminal(Mac OSX, Linux) 

	Windows: Press Win + R keys, type 'cmd' and press Enter

	Mac OSX: Go to Applications -> Utilities -> Terminal

	Linux: Navigate to terminal

2. Navigate to the folder containing the file "stockquotes.py" by typing 'cd [PATH_TO_FOLDER]' (for example: 'cd D:\Downloads\project\project' )

3. Run script by typing 

	There are two ways to run this script:

		a. Type 'python stockquotes.py', press Enter and follow the instructions.
			

		b. Type 'python stockquotes.py --date [YYYY-MM-DD]' and press Enter
